Games

WizFight is my main project currently. I am working on the game with Jordan Oakley. It is a 4-player side-scrolling deathmatch game that features four game modes, 20+ playable characters, 6+ stages, support for custom maps, and more in the works.
Snail Race Manager 2013 is an only semi-serious game about racing snails. The game was made for GBJam. The game has 3 modes of play, 3 snails to choose from, and online leaderboards as well as faux-multiplayer. It features music by Jordan Oakley.
Time Travelly Top Down Shooter is a game I work on from time to time. It's an arena shooter that employs a simple form of time travel.
